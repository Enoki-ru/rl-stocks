    4.  `sample()`: –ó–¥–µ—Å—å —Ä–µ–∞–ª–∏–∑—É–µ—Ç—Å—è **Reparameterization Trick**. –í–º–µ—Å—Ç–æ —Ç–æ–≥–æ —á—Ç–æ–±—ã —Å—ç–º–ø–ª–∏—Ä–æ–≤–∞—Ç—å –∏–∑ `N(Œº, œÉ)`, —á—Ç–æ –Ω–µ–¥–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏—Ä—É–µ–º–æ, –º—ã —Å—ç–º–ø–ª–∏—Ä—É–µ–º –∏–∑ `N(0, 1)` (—ç—Ç–æ –ø—Ä–æ—Å—Ç–æ —Å–ª—É—á–∞–π–Ω–æ–µ —á–∏—Å–ª–æ `Œµ`) –∏ –≤—ã—á–∏—Å–ª—è–µ–º –¥–µ–π—Å—Ç–≤–∏–µ –∫–∞–∫ `action = Œº + œÉ * Œµ`. –≠—Ç–æ –ø–æ–∑–≤–æ–ª—è–µ—Ç –≥—Ä–∞–¥–∏–µ–Ω—Ç–∞–º "–ø—Ä–æ—Ç–µ–∫–∞—Ç—å" —á–µ—Ä–µ–∑ `Œº` –∏ `œÉ` –∫ –æ—Å—Ç–∞–ª—å–Ω–æ–π —Å–µ—Ç–∏.
    5.  `torch.tanh`: –î–µ–π—Å—Ç–≤–∏–µ –ø—Ä–æ–ø—É—Å–∫–∞–µ—Ç—Å—è —á–µ—Ä–µ–∑ –≥–∏–ø–µ—Ä–±–æ–ª–∏—á–µ—Å–∫–∏–π —Ç–∞–Ω–≥–µ–Ω—Å, —á—Ç–æ–±—ã –æ–≥—Ä–∞–Ω–∏—á–∏—Ç—å –µ–≥–æ –∑–Ω–∞—á–µ–Ω–∏—è –¥–∏–∞–ø–∞–∑–æ–Ω–æ–º `[-1, 1]`. –≠—Ç–æ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–∞—è –ø—Ä–∞–∫—Ç–∏–∫–∞ –≤ SAC.
-   **–ü–æ—á–µ–º—É —Ç–∞–∫?** –ú—ã –º–æ–¥–µ–ª–∏—Ä—É–µ–º **—Å—Ç–æ—Ö–∞—Å—Ç–∏—á–µ—Å–∫—É—é –ø–æ–ª–∏—Ç–∏–∫—É**. –ê–≥–µ–Ω—Ç –Ω–µ –ø—Ä–æ—Å—Ç–æ –≤—ã–¥–∞–µ—Ç –æ–¥–Ω–æ –¥–µ–π—Å—Ç–≤–∏–µ, –∞ –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç —Ü–µ–ª–æ–µ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–µ–π –¥–ª—è –¥–µ–π—Å—Ç–≤–∏–π. –≠—Ç–æ –∫—Ä–∏—Ç–∏—á–µ—Å–∫–∏ –≤–∞–∂–Ω–æ –¥–ª—è –∏—Å—Å–ª–µ–¥–æ–≤–∞–Ω–∏—è (exploration) —Å—Ä–µ–¥—ã. `log_std` –∫–æ–Ω—Ç—Ä–æ–ª–∏—Ä—É–µ—Ç "—à–∏—Ä–∏–Ω—É" —ç—Ç–æ–≥–æ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è ‚Äî –Ω–∞—Å–∫–æ–ª—å–∫–æ –∞–≥–µ–Ω—Ç —É–≤–µ—Ä–µ–Ω –≤ —Å–≤–æ–∏—Ö –¥–µ–π—Å—Ç–≤–∏—è—Ö.
-   **–ê–ª—å—Ç–µ—Ä–Ω–∞—Ç–∏–≤—ã –∞–≥—Ä–µ–≥–∞—Ü–∏–∏?**
    -   `max()`: –í–∑—è—Ç–∏–µ –º–∞–∫—Å–∏–º—É–º–∞ –ø–æ –∏–∑–º–µ—Ä–µ–Ω–∏—è–º.
    -   **Attention Pooling:** –î–æ–±–∞–≤–∏—Ç—å –µ—â–µ –æ–¥–∏–Ω —Å–ª–æ–π –≤–Ω–∏–º–∞–Ω–∏—è, –∫–æ—Ç–æ—Ä—ã–π –±—ã –≤–∑–≤–µ—à–µ–Ω–Ω–æ —Å—É–º–º–∏—Ä–æ–≤–∞–ª —ç–º–±–µ–¥–¥–∏–Ω–≥–∏ –∞–∫—Ç–∏–≤–æ–≤ –≤ –æ–¥–∏–Ω –≤–µ–∫—Ç–æ—Ä. –≠—Ç–æ –±–æ–ª–µ–µ —Å–ª–æ–∂–Ω—ã–π, –Ω–æ –∏ –±–æ–ª–µ–µ –º–æ—â–Ω—ã–π –ø–æ–¥—Ö–æ–¥.

#### 1.4 `Critic` (Q-—Ñ—É–Ω–∫—Ü–∏—è)

-   **–ß—Ç–æ –¥–µ–ª–∞–µ—Ç?** –û—Ü–µ–Ω–∏–≤–∞–µ—Ç –ø–∞—Ä—É `(—Å–æ—Å—Ç–æ—è–Ω–∏–µ, –¥–µ–π—Å—Ç–≤–∏–µ)`. –û—Ç–≤–µ—á–∞–µ—Ç –Ω–∞ –≤–æ–ø—Ä–æ—Å: "–ï—Å–ª–∏ –≤ *—ç—Ç–æ–º* —Å–æ—Å—Ç–æ—è–Ω–∏–∏ —Å–æ–≤–µ—Ä—à–∏—Ç—å *—ç—Ç–æ* –¥–µ–π—Å—Ç–≤–∏–µ, –∫–∞–∫—É—é —Å—É–º–º–∞—Ä–Ω—É—é –Ω–∞–≥—Ä–∞–¥—É –º—ã –æ–∂–∏–¥–∞–µ–º –ø–æ–ª—É—á–∏—Ç—å –≤ –±—É–¥—É—â–µ–º?".

    2.  `torch.cat([aggregated_state, action], dim=1)`: –ö–æ–Ω–∫–∞—Ç–µ–Ω–∏—Ä—É–µ—Ç (–æ–±—ä–µ–¥–∏–Ω—è–µ—Ç) –≤–µ–∫—Ç–æ—Ä —Å–æ—Å—Ç–æ—è–Ω–∏—è –∏ –≤–µ–∫—Ç–æ—Ä –¥–µ–π—Å—Ç–≤–∏—è. –≠—Ç–æ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ, —Ç–∞–∫ –∫–∞–∫ Q-—Ñ—É–Ω–∫—Ü–∏—è –æ—Ü–µ–Ω–∏–≤–∞–µ—Ç –∏–º–µ–Ω–Ω–æ –ø–∞—Ä—É.
    3.  `net`: MLP –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞–µ—Ç –æ–±—ä–µ–¥–∏–Ω–µ–Ω–Ω—ã–π –≤–µ–∫—Ç–æ—Ä –∏ –≤—ã–¥–∞–µ—Ç –Ω–∞ –≤—ã—Ö–æ–¥–µ –æ–¥–Ω–æ —á–∏—Å–ª–æ ‚Äî **Q-value**.
-   **–ü–æ—á–µ–º—É —Ç–∞–∫?** –≠—Ç–æ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–∞—è —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è Q-—Ñ—É–Ω–∫—Ü–∏–∏ –≤ –ø–∞—Ä–∞–¥–∏–≥–º–µ Actor-Critic. –ö—Ä–∏—Ç–∏–∫ —É—á–∏—Ç—Å—è –ø—Ä–µ–¥—Å–∫–∞–∑—ã–≤–∞—Ç—å –±—É–¥—É—â–∏–µ –Ω–∞–≥—Ä–∞–¥—ã, –∞ –ê–∫—Ç–µ—Ä –∏—Å–ø–æ–ª—å–∑—É–µ—Ç —ç—Ç–∏ –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–∏—è, —á—Ç–æ–±—ã –Ω–∞—É—á–∏—Ç—å—Å—è –≤—ã–±–∏—Ä–∞—Ç—å –¥–µ–π—Å—Ç–≤–∏—è, –≤–µ–¥—É—â–∏–µ –∫ –≤—ã—Å–æ–∫–∏–º –æ—Ü–µ–Ω–∫–∞–º –ö—Ä–∏—Ç–∏–∫–∞.

    -   **–û–ø—Ç–∏–º–∏–∑–∞—Ç–æ—Ä:** –°–æ–∑–¥–∞–µ—Ç—Å—è *–µ–¥–∏–Ω—ã–π* `Adam` –æ–ø—Ç–∏–º–∏–∑–∞—Ç–æ—Ä, –∫–æ—Ç–æ—Ä–æ–º—É –ø–µ—Ä–µ–¥–∞—é—Ç—Å—è –ø–∞—Ä–∞–º–µ—Ç—Ä—ã **–≤—Å–µ—Ö** —Å–µ—Ç–µ–π (`Transformer`, `Actor`, `Critics`). –≠—Ç–æ –∏ –µ—Å—Ç—å —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è end-to-end –æ–±—É—á–µ–Ω–∏—è.

#### 2.2 `select_action`

-   **–ß—Ç–æ –¥–µ–ª–∞–µ—Ç?** –í—ã–±–∏—Ä–∞–µ—Ç –¥–µ–π—Å—Ç–≤–∏–µ –¥–ª—è –æ–¥–Ω–æ–≥–æ —Å–æ—Å—Ç–æ—è–Ω–∏—è (–∫–æ–≥–¥–∞ –∞–≥–µ–Ω—Ç –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤—É–µ—Ç —Å–æ —Å—Ä–µ–¥–æ–π).
-   **–ö–∞–∫ —Ä–∞–±–æ—Ç–∞–µ—Ç?**
    -   `state.unsqueeze(0)`: –î–æ–±–∞–≤–ª—è–µ—Ç —Ñ–∏–∫—Ç–∏–≤–Ω—É—é "batch" —Ä–∞–∑–º–µ—Ä–Ω–æ—Å—Ç—å, —Ç–∞–∫ –∫–∞–∫ —Å–µ—Ç–∏ –æ–∂–∏–¥–∞—é—Ç –Ω–∞ –≤—Ö–æ–¥ –±–∞—Ç—á.
    -   `with torch.no_grad()`: **–û—á–µ–Ω—å –≤–∞–∂–Ω–æ!** –û—Ç–∫–ª—é—á–∞–µ—Ç –≤—ã—á–∏—Å–ª–µ–Ω–∏–µ –≥—Ä–∞–¥–∏–µ–Ω—Ç–æ–≤. –ù–∞ —ç—Ç–æ–º —ç—Ç–∞–ø–µ –º—ã –ø—Ä–æ—Å—Ç–æ –∏—Å–ø–æ–ª—å–∑—É–µ–º –º–æ–¥–µ–ª—å, –∞ –Ω–µ –æ–±—É—á–∞–µ–º –µ–µ, –ø–æ—ç—Ç–æ–º—É –≥—Ä–∞–¥–∏–µ–Ω—Ç—ã –Ω–µ –Ω—É–∂–Ω—ã –∏ –∏—Ö —Ä–∞—Å—á–µ—Ç ‚Äî –ª–∏—à–Ω—è—è —Ç—Ä–∞—Ç–∞ —Ä–µ—Å—É—Ä—Å–æ–≤.
    -   `.cpu().squeeze(0).numpy()`: –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Ä–µ–∑—É–ª—å—Ç–∞—Ç –Ω–∞ CPU, —É–±–∏—Ä–∞–µ—Ç "batch" —Ä–∞–∑–º–µ—Ä–Ω–æ—Å—Ç—å –∏ –∫–æ–Ω–≤–µ—Ä—Ç–∏—Ä—É–µ—Ç –≤ NumPy –º–∞—Å—Å–∏–≤, –∫–æ—Ç–æ—Ä—ã–π –æ–±—ã—á–Ω–æ –æ–∂–∏–¥–∞—é—Ç —Å—Ä–µ–¥—ã `gymnasium`.

#### 2.3 –ì–ª—É–±–æ–∫–∏–π –†–∞–∑–±–æ—Ä `update`

–≠—Ç–æ —Å–µ—Ä–¥—Ü–µ –∞–ª–≥–æ—Ä–∏—Ç–º–∞. –ó–¥–µ—Å—å —Ä–µ–∞–ª–∏–∑—É—é—Ç—Å—è —Ñ–æ—Ä–º—É–ª—ã –∏–∑ **—Å—Ç–∞—Ç—å–∏ –ø–æ Soft Actor-Critic**.

**–®–∞–≥ 1: –í—ã—á–∏—Å–ª–µ–Ω–∏–µ Loss –¥–ª—è –ö—Ä–∏—Ç–∏–∫–æ–≤**
-   **–¢–µ–æ—Ä–∏—è:** –ö—Ä–∏—Ç–∏–∫ –¥–æ–ª–∂–µ–Ω –Ω–∞—É—á–∏—Ç—å—Å—è —É–¥–æ–≤–ª–µ—Ç–≤–æ—Ä—è—Ç—å —É—Ä–∞–≤–Ω–µ–Ω–∏—é –ë–µ–ª–ª–º–∞–Ω–∞. –í SAC –æ–Ω–æ –≤—ã–≥–ª—è–¥–∏—Ç —Ç–∞–∫:
    $Q(s, a) \approx r + \gamma \mathbb{E}_{s' \sim P, a' \sim \pi}[Q_{target}(s', a') - \alpha \log \pi(a'|s')]$
    –¶–µ–ª—å `y` (–≤ –∫–æ–¥–µ `q_target`) ‚Äî —ç—Ç–æ –ø—Ä–∞–≤–∞—è —á–∞—Å—Ç—å —É—Ä–∞–≤–Ω–µ–Ω–∏—è. Loss ‚Äî —ç—Ç–æ —Å—Ä–µ–¥–Ω–µ–∫–≤–∞–¥—Ä–∞—Ç–∏—á–Ω–∞—è –æ—à–∏–±–∫–∞ –º–µ–∂–¥—É –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–∏–µ–º `Q(s, a)` –∏ —Ü–µ–ª—å—é `y`.
-   **–†–µ–∞–ª–∏–∑–∞—Ü–∏—è –≤ –∫–æ–¥–µ:**
    1.  `with torch.no_grad()`: –í—Å–µ –≤—ã—á–∏—Å–ª–µ–Ω–∏—è –¥–ª—è `q_target` –Ω–µ –¥–æ–ª–∂–Ω—ã —Å–æ–∑–¥–∞–≤–∞—Ç—å –≥—Ä–∞–¥–∏–µ–Ω—Ç–æ–≤.
    2.  `next_action, next_log_pi = self.actor.sample(next_latent_state)`: –ü–æ–ª—É—á–∞–µ–º –¥–µ–π—Å—Ç–≤–∏–µ –¥–ª—è *—Å–ª–µ–¥—É—é—â–µ–≥–æ* —Å–æ—Å—Ç–æ—è–Ω–∏—è –æ—Ç *—Ç–µ–∫—É—â–µ–π* –ø–æ–ª–∏—Ç–∏–∫–∏.
    3.  `q1_target_next = self.critic1_target(...)`: –û—Ü–µ–Ω–∏–≤–∞–µ–º —ç—Ç–æ —Å–ª–µ–¥—É—é—â–µ–µ –¥–µ–π—Å—Ç–≤–∏–µ —Å –ø–æ–º–æ—â—å—é **—Ü–µ–ª–µ–≤—ã—Ö** —Å–µ—Ç–µ–π –ö—Ä–∏—Ç–∏–∫–∞.
    4.  `min_q_target_next = torch.min(q1_target_next, q2_target_next)`: **Double Q-learning!** –ú—ã –∏—Å–ø–æ–ª—å–∑—É–µ–º –º–∏–Ω–∏–º—É–º –∏–∑ –¥–≤—É—Ö –æ—Ü–µ–Ω–æ–∫. –≠—Ç–æ –ø–æ–º–æ–≥–∞–µ—Ç –±–æ—Ä–æ—Ç—å—Å—è —Å –∑–∞–≤—ã—à–µ–Ω–∏–µ–º Q-–∑–Ω–∞—á–µ–Ω–∏–π, –∫–æ—Ç–æ—Ä–æ–µ —è–≤–ª—è–µ—Ç—Å—è –∏–∑–≤–µ—Å—Ç–Ω–æ–π –ø—Ä–æ–±–ª–µ–º–æ–π –≤ RL.
    5.  `q_target = reward + self.gamma * (1 - done) * (min_q_target_next - alpha * next_log_pi)`: –°–æ–±–∏—Ä–∞–µ–º —Ü–µ–ª–µ–≤–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ. `(1-done)` –æ–±–Ω—É–ª—è–µ—Ç –±—É–¥—É—â—É—é –Ω–∞–≥—Ä–∞–¥—É, –µ—Å–ª–∏ —ç–ø–∏–∑–æ–¥ –∑–∞–∫–æ–Ω—á–∏–ª—Å—è. `alpha * next_log_pi` ‚Äî —ç—Ç–æ –±–æ–Ω—É—Å –∑–∞ —ç–Ω—Ç—Ä–æ–ø–∏—é.
    6.  `critic_loss = F.mse_loss(q1_current, q_target) + ...`: –í—ã—á–∏—Å–ª—è–µ–º MSE loss.
-   **–§–æ—Ä–º—É–ª–∞ Loss –ö—Ä–∏—Ç–∏–∫–∞:** $L_{critic} = \mathbb{E}_{(s,a,r,s') \sim D} \left[ (Q_1(s,a) - y)^2 + (Q_2(s,a) - y)^2 \right]$, –≥–¥–µ $y = r + \gamma (1-d) (\min_{i=1,2} Q'_{i}(s', a') - \alpha \log \pi(a'|s'))$

**–®–∞–≥ 2: –í—ã—á–∏—Å–ª–µ–Ω–∏–µ Loss –¥–ª—è –ê–∫—Ç–µ—Ä–∞**
-   **–¢–µ–æ—Ä–∏—è:** –ê–∫—Ç–µ—Ä –¥–æ–ª–∂–µ–Ω –Ω–∞—É—á–∏—Ç—å—Å—è –≤—ã–±–∏—Ä–∞—Ç—å –¥–µ–π—Å—Ç–≤–∏—è, –∫–æ—Ç–æ—Ä—ã–µ –º–∞–∫—Å–∏–º–∏–∑–∏—Ä—É—é—Ç –æ—Ü–µ–Ω–∫—É –ö—Ä–∏—Ç–∏–∫–∞ –∏ –æ–¥–Ω–æ–≤—Ä–µ–º–µ–Ω–Ω–æ –º–∞–∫—Å–∏–º–∏–∑–∏—Ä—É—é—Ç —ç–Ω—Ç—Ä–æ–ø–∏—é (—Å–ª—É—á–∞–π–Ω–æ—Å—Ç—å).
    $J_{\pi}(\phi) = \mathbb{E}_{s \sim D, a \sim \pi_{\phi}}[\min_{i=1,2} Q_i(s, a) - \alpha \log \pi_{\phi}(a|s)]$
-   **–†–µ–∞–ª–∏–∑–∞—Ü–∏—è –≤ –∫–æ–¥–µ:**
    1.  `pi, log_pi = self.actor.sample(latent_state)`: –ü–æ–ª—É—á–∞–µ–º –Ω–æ–≤–æ–µ –¥–µ–π—Å—Ç–≤–∏–µ –∏ –µ–≥–æ –ª–æ–≥-–≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –¥–ª—è *—Ç–µ–∫—É—â–µ–≥–æ* —Å–æ—Å—Ç–æ—è–Ω–∏—è.
    2.  `min_q_pi = torch.min(self.critic1(...), self.critic2(...))`: –û—Ü–µ–Ω–∏–≤–∞–µ–º —ç—Ç–æ –Ω–æ–≤–æ–µ –¥–µ–π—Å—Ç–≤–∏–µ —Å –ø–æ–º–æ—â—å—é *–æ—Å–Ω–æ–≤–Ω—ã—Ö* (–Ω–µ —Ü–µ–ª–µ–≤—ã—Ö!) —Å–µ—Ç–µ–π –ö—Ä–∏—Ç–∏–∫–∞.
    3.  `actor_loss = ((alpha * log_pi) - min_q_pi).mean()`: –§–æ—Ä–º–∏—Ä—É–µ–º loss. –ú—ã –µ–≥–æ –º–∏–Ω–∏–º–∏–∑–∏—Ä—É–µ–º, –ø–æ—ç—Ç–æ–º—É `min_q_pi` –∏–¥–µ—Ç —Å–æ –∑–Ω–∞–∫–æ–º –º–∏–Ω—É—Å (–º–∏–Ω–∏–º–∏–∑–∞—Ü–∏—è `-Q` —ç–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω–∞ –º–∞–∫—Å–∏–º–∏–∑–∞—Ü–∏–∏ `Q`). `alpha * log_pi` –∏–¥–µ—Ç —Å–æ –∑–Ω–∞–∫–æ–º –ø–ª—é—Å (–º–∏–Ω–∏–º–∏–∑–∞—Ü–∏—è `alpha * log_pi` —ç–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω–∞ –º–∞–∫—Å–∏–º–∏–∑–∞—Ü–∏–∏ `-log_pi`, —Ç.–µ. –º–∞–∫—Å–∏–º–∏–∑–∞—Ü–∏–∏ —ç–Ω—Ç—Ä–æ–ø–∏–∏).
-   **–§–æ—Ä–º—É–ª–∞ Loss –ê–∫—Ç–µ—Ä–∞:** $L_{actor} = \mathbb{E}_{s \sim D, \epsilon \sim N} [ \alpha \log \pi(f(\epsilon; s)|s) - \min_{i=1,2} Q_i(s, f(\epsilon; s)) ]$

**–®–∞–≥ 3: –í—ã—á–∏—Å–ª–µ–Ω–∏–µ Loss –¥–ª—è Alpha**
-   **–¢–µ–æ—Ä–∏—è:** –ú—ã —Ö–æ—Ç–∏–º, —á—Ç–æ–±—ã —Å—Ä–µ–¥–Ω—è—è —ç–Ω—Ç—Ä–æ–ø–∏—è –ø–æ–ª–∏—Ç–∏–∫–∏ –±—ã–ª–∞ –Ω–µ –Ω–∏–∂–µ —Ü–µ–ª–µ–≤–æ–≥–æ –∑–Ω–∞—á–µ–Ω–∏—è `target_entropy`.
    $J(\alpha) = \mathbb{E}_{a \sim \pi}[-\alpha \log \pi(a|s) - \alpha \mathcal{H}]$
-   **–†–µ–∞–ª–∏–∑–∞—Ü–∏—è –≤ –∫–æ–¥–µ:**
    1.  `alpha_loss = -(self.log_alpha * (log_pi + self.target_entropy).detach()).mean()`: `.detach()` –∑–¥–µ—Å—å –∫—Ä–∏—Ç–∏—á–µ—Å–∫–∏ –≤–∞–∂–Ω–æ. –ú—ã –Ω–µ —Ö–æ—Ç–∏–º, —á—Ç–æ–±—ã –≥—Ä–∞–¥–∏–µ–Ω—Ç—ã –æ—Ç —ç—Ç–æ–≥–æ loss'–∞ —Ç–µ–∫–ª–∏ –≤ —Å–µ—Ç—å –ê–∫—Ç–µ—Ä–∞. –ú—ã —Ö–æ—Ç–∏–º –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å `log_pi` —Ç–æ–ª—å–∫–æ –∫–∞–∫ –∫–æ–Ω—Å—Ç–∞–Ω—Ç—É –¥–ª—è –≤—ã—á–∏—Å–ª–µ–Ω–∏—è "–æ—à–∏–±–∫–∏" —Ç–µ–∫—É—â–µ–π –∞–ª—å—Ñ—ã.
-   **–§–æ—Ä–º—É–ª–∞ Loss Alpha:** $L_{\alpha} = \mathbb{E}_{a \sim \pi} [-\alpha (\log \pi(a|s) + \mathcal{H})]$

**–®–∞–≥ 4: –û–±–Ω–æ–≤–ª–µ–Ω–∏–µ –í–µ—Å–æ–≤**
-   **–¢–µ–æ—Ä–∏—è:** –ü—Ä–∏–º–µ–Ω—è–µ–º –≥—Ä–∞–¥–∏–µ–Ω—Ç–Ω—ã–π —Å–ø—É—Å–∫.
-   **–†–µ–∞–ª–∏–∑–∞—Ü–∏—è –≤ –∫–æ–¥–µ:**
    1.  `optimizer.zero_grad()`: –û—á–∏—â–∞–µ–º —Å—Ç–∞—Ä—ã–µ –≥—Ä–∞–¥–∏–µ–Ω—Ç—ã.
    2.  `(critic_loss + actor_loss).backward()`: –í—ã—á–∏—Å–ª—è–µ–º –≥—Ä–∞–¥–∏–µ–Ω—Ç—ã –¥–ª—è –≤—Å–µ—Ö —Å–µ—Ç–µ–π –æ—Ç —Å—É–º–º–∞—Ä–Ω–æ–π –æ—à–∏–±–∫–∏.
    3.  `optimizer.step()`: –û–±–Ω–æ–≤–ª—è–µ–º –≤–µ—Å–∞ –¢—Ä–∞–Ω—Å—Ñ–æ—Ä–º–µ—Ä–∞, –ê–∫—Ç–µ—Ä–∞ –∏ –ö—Ä–∏—Ç–∏–∫–æ–≤.
    4.  **"–ú—è–≥–∫–æ–µ" –æ–±–Ω–æ–≤–ª–µ–Ω–∏–µ —Ü–µ–ª–µ–≤—ã—Ö —Å–µ—Ç–µ–π:** `target_param.data.mul_(1.0 - self.tau) ...`. –≠—Ç–æ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è —Ñ–æ—Ä–º—É–ª—ã **Polyak Averaging**: $\theta' \leftarrow \tau\theta + (1-\tau)\theta'$. `Œ∏'` ‚Äî –≤–µ—Å–∞ —Ü–µ–ª–µ–≤–æ–π —Å–µ—Ç–∏, `Œ∏` ‚Äî –≤–µ—Å–∞ –æ—Å–Ω–æ–≤–Ω–æ–π. `œÑ` ‚Äî –æ—á–µ–Ω—å –º–∞–ª–µ–Ω—å–∫–æ–µ —á–∏—Å–ª–æ (0.005), –æ–±–µ—Å–ø–µ—á–∏–≤–∞—é—â–µ–µ –º–µ–¥–ª–µ–Ω–Ω–æ–µ, —Å—Ç–∞–±–∏–ª—å–Ω–æ–µ –æ–±–Ω–æ–≤–ª–µ–Ω–∏–µ.
